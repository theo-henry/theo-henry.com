# theo-henry.com

Theo Henry's personal portfolio website. It presents his experience, projects, certifications, education, and contact information.

## Built With

This is a static website built with plain HTML, CSS, and JavaScript. The editable source lives in `index.html`, `styles.css`, `app.js`, and `portfolio-data.js`.

There is no build step. Netlify serves the production site from the static publish directory configured in `netlify.toml`.

## Access

Public site: https://theo-henry.com

For local preview:

```sh
npm run dev
```

Then open http://localhost:5173.
